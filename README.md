# Cybersecurity Learning

This repository documents my structured learning journey in cybersecurity as a BSc IT student.

It includes notes, explanations, and hands-on lab writeups covering Linux fundamentals, networking concepts, and basic security practices. The goal of this repository is to build strong foundations and demonstrate practical understanding through real examples.

## Contents

- Linux basics (permissions, users, services, logs)
- Networking fundamentals (TCP/IP, DNS, HTTP, ports)
- Security fundamentals (common vulnerabilities and misconfigurations)
- TryHackMe lab notes and writeups

## Platforms & Tools
- Linux
- Nmap
- Burp Suite (basic)
- TryHackMe
- Git & GitHub

## Note
All practice is done in legal, controlled environments for learning purposes only.
